
# ML Toolbox

A collection of machine learning tools and experiments for reference.

## Contents

### Computer Vision
- **CNN**: Keras CNN implementation and VGG16 models
    - `cnn_keras.ipynb`
    - `vgg16.ipynb`
- **Image Processing**: Image manipulation and transformation
    - `Image Processing Test.ipynb`
    - `Photo to Sketch.ipynb`

### General
- **Chatbot**: Chatbot implementation with server
    - `botwork.py`
    - `server.py`
    - `test_server.py`
    - `requirements.txt`
- **TFRecord**: TensorFlow record handling for tabular data
    - `tabular_tfrecord.ipynb`

### TensorFlow
- **Basics**: TensorFlow fundamentals
    - `tf_basics.ipynb`

## Getting Started

Each directory contains its own experiments and utilities. Refer to individual `README.md` files for specific setup instructions.
